Implementácia

Aplikáciu sme implementovali v jazyku PHP s použitím frameworku Laravel v10. Na frontend sme použili React.

  • app/Http/Controllers/UserController.php: správa užívateľov
  • app/Http/Controllers/AuthController.php: správa sedení
  • app/Http/Controllers/EventController.php: správa udalostí
  • app/Http/Controllers/CategoryController.php: správa kategorií udalostí
  • app/Http/Controllers/LocationController.php: správa lokácií
  • app/Http/Controllers/TicketController.php: správa listkov pre udalosti
  • app/Http/Controllers/ReviewController.php: správa hodnotení udalostí

    • Databáza

    Na inicializaciu databazy sme pouzili zabudovanu funkcionalitu frameworku Laravel, migracie a databazove seedre.

    Inštalácia

    Prerekvizity

    • PHP 8.1
    • Composer 2.2.6
    • npm 10.1.0
    • Node.js 20.1.0
    • Databáza, napríklad MySQL

    Lokálna inštalácia

    Pre lokálnu inštaláciu je tu link na GitHub: https://github.com/tigrisske/IIS-projekt. Je potrebné spustiť Backend a Frontend časti aplikácie.

    Backend

    1. V koreňovom adresári projektu spustením príkazu composer install nainštalujeme potrebné balíčky.
    2. Vytvoriť a nakonfiigurovať súbor .env (príklad v .env.example)
    3. php artisan migrate:refresh --seed vytvoríme potrebné tabuľky v databáze a naplníme ich testovacími dátami.
    4. php artisan serve spustíme lokálny server.

    Frontend

    1. V adresári react spustením npm install nainštalujeme potrebné balíčky.
    2. Vytvoriť a nakonfiigurovať súbor .env (príklad v .env.example)
    3. npm start spustíme lokálny server.

    Online inštalácia

    Aplikáciu sme nasadili na server Hostinger. Je dostupná na adrese https://laravel-iis.com/.

    Známé problémy

    Užívateľ pri vytváraní eventu nieje schopný pridať svoju vlastnú fotku. V malom množste prípadov pri prístupe na server sa zobrazí chyba CORS konfigurácie.
